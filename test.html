Git és a GitHub használatakor felmerülhet Önben, hogy mi a különbség a kettő között.

A Git egy elosztott verziókövetési rendszer (DVCS), amellyel több fejlesztő vagy más közreműködő is dolgozhat egy projekten.

Módszert biztosít egy vagy több helyi ág használatára, majd azok leküldésére egy távoli adattárba. A Git kezel mindent a GitHubbal kapcsolatban, ami helyileg történik a számítógépen. A Git által biztosított főbb funkciók a következők:
  • A helyi gépen van telepítve és használva
  • Kezeli a verziókövetést
  • Támogatja az elágaztatást
